#### 配合pm2-web实现监控可视化
```
npm install -g pm2-web
```
默认pm2-web会自动启动一个端口8080，但是我们还是喜欢可控状态的，因此按照配置文件的方式启动。
```
pm2-web --config pm2-web-config.json
```
pm2-web-config.json
```
{
  "www": {
      "host": "localhost",
      "address": "0.0.0.0",
      "port": 6688
  }                         
}
```
